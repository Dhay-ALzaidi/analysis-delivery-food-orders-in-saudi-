# 📊 Analysis of Food Delivery Orders in Saudi Arabia

## 🎯 Project Overview
This project analyzes 10,000 food delivery records across multiple Saudi cities to evaluate customer behavior, restaurant performance, and delivery efficiency using **Python** and **Power BI**.

---

## 🛠️ Repository Structure
All project assets are organized in the folders above:
* 📂 **Basic data file (before cleaning)**: Raw delivery dataset.
* 📂 **Data file (after cleaning)**: Cleaned dataset using Python.
* 📂 **code**: Python scripts used for data cleaning and exploration.
* 📂 **dashboard**: The interactive Power BI dashboard source file (`.pbix`).
* 📂 **Results and recommendations**: Contains the full **PDF Report** with all dashboard screenshots, key insights, and final recommendations.
